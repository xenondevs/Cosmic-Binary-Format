# Cosmic-Binary-Format

A binary serialization library primarily used in [Nova](https://github.com/xenondevs/Nova).

[Documentation](https://docs.xenondevs.xyz/cbf/), [KDoc](https://cbf.dokka.xenondevs.xyz/)

## Maven

```xml
<repository>
    <id>xenondevs</id>
    <url>https://repo.xenondevs.xyz/releases/</url>
</repository>
```

```xml
<dependency>
    <groupId>xyz.xenondevs.cbf</groupId>
    <artifactId>cosmic-binary-format</artifactId>
    <version>VERSION</version>
</dependency>
```

## Gradle

```kotlin 
maven("https://repo.xenondevs.xyz/releases/")
```

```kotlin
implementation("xyz.xenondevs.cbf:cosmic-binary-format:VERSION")
```
